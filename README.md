# dnn-steel

This repository contains the Python code used to predict the yield strength of steels based on chemical composition using machine learning and deep learning models.

The implementation includes linear regression, random forest, support vector regression (SVR), XGBoost, and a deep neural network (DNN), along with interpretability analysis using SHAP and permutation feature importance.
